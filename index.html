Plex



O Plex é um media player que permite fazer stream do nosso conteúdo, quer sejam filmes, música ou fotos. Para que isto aconteça é necessário associarmos, numa conta, e utilizando o Plex Media Server, as pastas que queremos colocar na nossa network. Este conteúdo pode ser acessado posteriormente a partir de qualquer dispositivo ( tele, tablet,pc etc.), a partir da aplicação disponibilizada na app store ( ou, no caso do pc, no website do plex).
